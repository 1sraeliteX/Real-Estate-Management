# Documentation

## Structure

- **guides/**: User guides, quick start guides, and tutorials
- **implementation/**: Implementation progress, migration notes, and setup documentation

- `BEST_PRACTICES.md`: Development best practices

- `EXAMPLE_API_USAGE.md`: API usage examples
