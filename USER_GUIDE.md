## üè¢ Managing Properties

### Viewing Properties

1. Click on **"Properties"** card on the dashboard
2. Choose between:
   - **On-Campus Properties** (blue card) - Student lodges with shared facilities
   - **Off-Campus Properties** (purple card) - Independent apartments and houses

### Adding a New Property

1. Go to the property category page (On-Campus or Off-Campus)
2. Click **"Add Property"** button (top-right)
3. Fill in the details:
   - Property Name (e.g., "Ikechukwu Lodge")
   - Address (e.g., "Along Holy Trinity Catholic Church, Hilltop, Nsukka")
   - Number of rooms/bedrooms
   - Yearly rent
   - Upload property images
4. Click **"Create Property"**

### Viewing Property Details

1. Click on any property card
2. You'll see:
   - Property image at the top
   - Property information
   - Four tabs: **Rooms**, **Occupants**, **Finance**, **Maintenance**

### Managing Rooms

1. Go to property details
2. Click **"Rooms"** tab
3. Click **"Add Room"** button
4. Enter room number and yearly rent
5. After adding, you can immediately add an occupant

**Room Cards Show:**
- Room number
- Status (Available/Occupied)
- Yearly rent
- Number of occupants
- Payment status (‚úì Complete or ‚è≥ Pending)

## üë• Managing Occupants

### Viewing Occupants

1. Click on **"Occupants"** card on the dashboard
2. Choose between:
   - **On-Campus Occupants** (blue card)
   - **Off-Campus Occupants** (purple card)

### Adding an Occupant

**Method 1: From Property Details**
1. Go to property details ‚Üí Occupants tab
2. Click **"Add Occupant"** button
3. Select a room
4. Fill in occupant details

**Method 2: After Adding a Room**
1. When you add a room, a success message appears
2. Click **"Add Occupant"** button
3. Fill in the details

### Contacting Occupants

Each occupant card has two buttons:
- **WhatsApp** (green) - Opens WhatsApp chat
- **Call** (blue) - Initiates phone call

### Filtering Occupants

Use the filters at the top:
- **Search bar** - Search by name, property, room, or phone
- **Property dropdown** - Filter by specific property
- **Status buttons** - Filter by payment status (All/Paid/Pending)

## üí∞ Managing Finances

### Viewing Finances

1. Click on **"Finances"** card on the dashboard
2. Choose between:
   - **On-Campus Finances** (green card)

### Understanding Finance Stats

Each finance page shows:
- **Total Revenue** - Money collected
- **Total Expected** - Total amount due
- **Pending Amount** - Awaiting payment
- **Overdue Amount** - Late payments

### Property Breakdown Table

Shows financial summary for each property:
- Collected amount (green)
- Pending amount (yellow)
- Overdue amount (red)
- Total

### Exporting Finance Data

1. Click **"Export"** button (top-right)
2. Optionally select date range
3. Click **"Export CSV"**
4. File downloads automatically

## üÜò Common Questions

**Q: How do I add multiple rooms at once?**
A: Currently, add rooms one at a time. After adding each room, you can quickly add an occupant.

**Q: Can I edit property information?**
A: Yes, click the "Edit" button on any property card.

**Q: How do I mark a payment as complete?**
A: Edit the occupant details and update the payment status.

**Q: Can I delete a property?**
A: Yes, click the "Delete" button on the property card. This action cannot be undone.

**Q: How do I export all data?**
A: Go to the Finance page and click "Export" to download a CSV file.
